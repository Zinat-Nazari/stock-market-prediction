# stock-market-prediction
# Stock Market Prediction using Machine Learning
this project predicts stock market direction (Up/Down) using machine learning models and historical stock data.
---
## Project Overview
the goal of this project is to predict whether a stock price will go up or down using historical data and machine learning techniques.
---
## Features Used
- Daily Returns
- Moving Average (MA5 , MA10)
- Lag Returns
- Volatility
- Volume Changes
---
## Models Used
- Logestic Regression
- Random Forest Classifier
---
## Evaluation Matrics
- Accuracy Score
- Confusion Matrix
- Classification Report
---
## Visualizations
- Stock Price Trend
- Prediction Vs Actual
- Feature Importance
---
## Key Insight
Random Forest performed better than Logistic Regression due to its ability to capture nonlinear relationships in financial data.
